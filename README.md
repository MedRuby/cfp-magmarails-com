## Basic Configuration

###ENV VARIABLES

- DEVISE_SECRET_KEY
  ```rake secret``` would do the trick

- CFP_APP_ID
  Github App Id

- CFP_APP_SECRET
  Github App Secret

- CFP_ENDING_DATE
  Last date for paper submission. Format: yyy-mm-dd
